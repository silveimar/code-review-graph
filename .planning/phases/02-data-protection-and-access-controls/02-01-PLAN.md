<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| env→key loader | Untrusted host supplies key material |
| disk→decrypt | Local ciphertext crosses into plaintext in process memory |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | I | `artifact_crypto.py` key handling | mitigate | Load key only from named env; never persist key to disk; clear buffers where feasible |
| T-02-02 | T | encrypted JSONL / markdown | mitigate | Fernet provides integrity; reject tampered ciphertext on decrypt |
| T-02-03 | E | missing encryption key | mitigate | Fail-closed: refuse sensitive I/O and surface explicit error per D-03 |
| T-02-04 | E | `GraphStore` fallback | mitigate | No silent plaintext DB when policy requires encryption |
</threat_model>
